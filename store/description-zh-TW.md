# PDF 數位簽章驗證器

在瀏覽器中直接驗證 PDF 數位簽章 — 無需上傳檔案、無需外部服務。所有驗證皆在本機執行，確保最高隱私安全。

## 功能特色

- **完整性驗證** — 確認 PDF 在簽署後未被竄改（CMS/PKCS#7 摘要 + 密碼學簽章驗證）
- **憑證鏈驗證** — 建立並驗證從簽署者到根憑證的完整 X.509 憑證鏈
- **信任根驗證** — 檢查簽署憑證是否連結到受信任的根 CA（支援全球 CA 及台灣 PKI：TWCA、GCA、MOICA）
- **RFC 3161 時戳驗證** — 驗證內嵌時戳，證明文件的簽署時間
- **憑證有效期檢查** — 確認憑證在簽署時是否有效，支援 LTV 回溯驗證已過期憑證
- **撤銷狀態檢查** — 透過內嵌或線上 OCSP 及 CRL 確認憑證未被撤銷
- **長期驗證 (LTV)** — 檢查簽章是否包含足夠的內嵌資料以供長期驗證
- **詳細報告匯出** — 匯出完整的 PDF 驗證報告

## 使用方式

1. 開啟任意 PDF 檔案，或點擊網頁上 PDF 連結旁的驗證按鈕
2. 擴充功能自動偵測並驗證所有數位簽章
3. 查看詳細結果，包括憑證鏈、時戳、撤銷狀態
4. 可匯出完整驗證報告為 PDF

## 隱私保護

所有驗證皆在瀏覽器本機執行，您的 PDF 檔案絕不會上傳至任何伺服器。

## 支援的簽章格式

- CMS / PKCS#7（adbe.pkcs7.detached、adbe.pkcs7.sha1）
- RSA、ECDSA、RSA-PSS 簽章演算法
- SHA-256、SHA-384、SHA-512、SHA-1 雜湊演算法
- PAdES（PDF 進階電子簽章）

## 語言支援

繁體中文及英文。自動偵測瀏覽器語言，亦可在設定中手動切換。
